# Prism
An open sourced framework for easily building discord bots

# Installation
1. **Use Python 3.6 or newer**
2. **Install dependencies if needed**
3. **Create .env file according to this template:**

```
TOKEN=<Bot Token>
BOT_PREFIX=<Bot Prefix>
```
